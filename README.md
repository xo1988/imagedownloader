# image downloader
> 一款方便查看下载图片的插件

# 安装
* [chrome web store](https://chrome.google.com/webstore/detail/image-downloader-pro/mbemenikbjakghfmdkfdochjdknoohab)

# 功能
* 支持根据图片格式筛选显示图片
* 支持鼠标滚动浏览图片
* 支持缩略图预览图片
* 支持快速定位到图片位置
* 支持快速下载当前图片
* 支持批量下载图片
* 支持显示文件名称及尺寸信息
